    Layout

    Layout

    (DO NOT USE) Class with the layout (vertex stride, etc.) of 1 vertex.

    Constructor

    # new Layout()

    (DO NOT USE) Internal use by Hummingbird only, sets HB.vertexStride to 0.

    Methods

    # add(name, type, count, normalized)

    (DO NOT USE) Add a new attribute for rendering a vertex.
    Parameters:
    Name Type Default Description
    name string Name of attribute in the shader.
    type GLenum Type of data, see MDN.
    count number Amount of bytes for this attribute.
    normalized boolean false Whether integers should be normalized into a certain range when being cast to a float, https://developer.mozilla.org/en-US/docs/Web/API/WebGLRenderingContext/vertexAttribPointer#parameters.
